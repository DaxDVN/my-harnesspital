# second-brain raw-note archive

This folder stores raw `YYYY-MM-DD-*.md` learning notes after they have been compressed into `../grouped/`.

Default agent behavior:

- read `../grouped/` first;
- do not scan this archive wholesale;
- open an archived raw note only when a grouped rule needs provenance, exact evidence, or promotion detail.

New uncompressed learnings should still be captured in `second-brain/` root first. Periodic maintenance can fold
them into `grouped/` and then move the raw note here.
